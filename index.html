Connect to Google Maps to find real EV stations

Follow these 4 steps — takes about 5 minutes and is free to get started.

1
Create a Google Cloud account
Go to console.cloud.google.com and sign in with Google. New accounts get $300 free credit — you won't be charged just for testing.
2
Create a new project
Click the project dropdown at the top → New Project → name it EVMumbai → click Create. Wait ~10 seconds for it to be ready.
3
Enable 3 APIs
Go to APIs & Services → Library and search for and enable each of these:

• Maps JavaScript API
• Places API
• Directions API
4
Get your API key and paste it below
Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key (starts with AIza) and paste it here.
🔒 Stored only in your browser. Never sent to any server.
